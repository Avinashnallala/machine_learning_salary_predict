# Salary Prediction using Machine Learning

## Overview

This project predicts salaries based on years of experience using Linear Regression.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Machine Learning Model

* Linear Regression

## Dataset Features

* YearsExperience
* Salary

## Workflow

1. Data Preprocessing
2. Train-Test Split
3. Model Training
4. Prediction
5. Visualization

## Output

The model predicts employee salary based on experience.
